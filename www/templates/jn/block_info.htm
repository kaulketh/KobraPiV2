what kind of project

As a hobby, I deal with 3D printing, programming, microcontrollers and single-board computers and plan to use this server here to control and monitor a 3D printer. It already runs a power consumption monitor including automatic shutdown of the printer, a temperature monitor including fan control for the server and this self programmed website here in a flask-gunicorn framework.
Sockets switchable via Wi-Fi, flashed with {% set lurl, ltext="https://bit.ly/tasmota", "Tasmota" %} {% include 'jn/block_link_blank.htm' %} to control power and for consumption measurement, and self-built Esp32 Wi-Fi cameras are also used.
I have implemented the usage of the {% set lurl, ltext="https://telegram.org/", "Telegram" %} {% include 'jn/block_link_blank.htm' %} bot API to monitor and control printer and server.
I like Python, taught myself and use Pycharm as IDE and {% set lurl, ltext="https://github.com/", "GitHub" %} {% include 'jn/block_link_blank.htm' %} as code repository.
